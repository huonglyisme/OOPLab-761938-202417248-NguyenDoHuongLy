1. What happens if users choose "Cancel" in ChoosingOption?
With the original code, if users choose "Cancel", the program shows "No".
This happens because the code only checks whether the selected option is YES_OPTION.
All other cases, including NO_OPTION and CANCEL_OPTION, are treated as "No".

2. How to customize the options to users?
There are two common ways:
- Use JOptionPane.showConfirmDialog(..., JOptionPane.YES_NO_OPTION) to display only "Yes" and "No".
- Use JOptionPane.showOptionDialog(...) to create custom button labels such as "I do" and "I don't".
